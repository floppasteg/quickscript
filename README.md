# Pseudocode that can run

What if you want to make the next [insert program here], but hate having to use the darn shift key all the time? No?

What about a modern day equivalent to COBOL? Where the code is semi-English and (almost) anyone who can code can read it.

No reference implementation yet, PRs welcome. (preferably in Go or C)
